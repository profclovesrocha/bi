Padrões de projeto aplicados
🔌
Adapter
Cada subsistema tem um adaptador que converte o formato proprietário (JSON, CSV, XML) para o schema canônico sem alterar o sistema original.
🔄
Observer / Event Bus
Mudanças em um subsistema disparam eventos no EventBus. Os demais reagem automaticamente — sem acoplamento direto.
🎯
Strategy
Os adaptadores são intercambiáveis. Trocar o fornecedor de tráfego, por exemplo, não exige alteração no CityHub.
💉
Dependency Injection
O CityHub recebe os adaptadores via construtor, facilitando testes unitários com mocks isolados.
