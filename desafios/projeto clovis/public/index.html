Padrões de Projeto Aplicados
🔌
Adapter
Converte formatos proprietários (JSON, CSV, XML) para o schema canônico sem alterar os sistemas originais.
📡
Observer / Event Bus
Mudanças em um subsistema disparam eventos. Os demais reagem automaticamente — sem acoplamento direto.
🎯
Strategy
Adaptadores são intercambiáveis. Trocar o fornecedor não exige modificação no CityHub.
💉
Dependency Injection
CityHub recebe adaptadores via construtor, permitindo testes unitários com mocks isolados.

Regras de Automação do CityHub
🌧️
Chuva ou tempestade detectada pelo sistema meteorológico
Iluminação → mín. 80%
🚗
Congestionamento acima de 80% em alguma via monitorada
Iluminação → 100%
🚦
Congestionamento acima de 70% na via (ajuste automático)
Base + 30%
☀️
Luminosidade natural acima de 70% (pleno dia)
Base − 20%
